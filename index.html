Wayne's HTML/CSS Cheat Sheet

HTML

Table Tags & Attributes

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Media & Others

Tag Name Example / Description
<img> Image <img src="image.png" alt="image description">
<video> Video <video src="test-video.mp4" controls> Video not supported </video>
<audio> Audio <audio src="koreanhiphop.mp3" controls></audio>
<span> Span Used in <p> tags to style text.

CSS

Selectors

Property Name Example / Description
.example h2 Descendant Selector Selects the child of a parent element.
h2.example Chaining Selector Selects the element that fits all criteria.
h1, h2 Group Selector Selects multiple elements to apply style to.
#example ID Selector Selection for an ID element.

Display & Positioning

Property Name Example / Description
{position: relative;} Relative Allows an element to be positioned relative to its original place.
{position: absolute;} Absolute Removes element from document flow.
{postion : fixed;} Fixed Locks an element to the page.
{position: sticky;} Sticky Locked into a position until coming into contact with an ancestor block.
{float: left;} Float Determines how far left/right an element goes within its parent element.
{clear: left;} Clear Specifies how an element behaves when bumping into element in the same container.
{z-index: 1;} Z-Index A property that specifies how front or back an element is placed.

For more information refer to Codecademy's Cheat Sheet for HTML Tables & Media and CSS Selectors, Display & Positioning.
